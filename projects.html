Productivity Stopwatch

Created March 12, 2024

A simple GUI stopwatch with pausing, logging, and data analysis capabilities. Written in Python using Tkinter and Pandas.

I needed a reliable stopwatch for tracking time spent working on projects, both paid and personal. More importantly, I needed a way to log that time, including any breaks and pauses.
This project is the solution to such an issue, and provides a reliable time-tracking solution for those of us who like to keep track of their temporal expenditure.

(Top) The GUI of the stopwatch, with text entries for title, subtitle, and an additional variable
(Bottom) An example of the log created by the stopwatch

Bank Statement Analyzer

Created July 20, 2024

A Python script that allows the user to scrape transaction information from bank statements from a number of banks and account types.

This project is a single (as of now) Python script that provides options for the user that involve scraping transactions and their information straight from PDF bank statements. The scraped data is stored in multiple CSV files, one for each account as well as compiled supersets of entire banks and all data.
The script uses JSON storage to keep track of which statements have been read and if the compiled data has been modified, and is able to update the data with new statements accordingly.

(Top) The options available to the user in the CLI, allowing updating, recompiling, or viewing of scraped data
(Bottom) Info for the DataFrame of compiled transactions retrieved using the script

Python Minesweeper

Created February 24, 2024

A simple implementation of the classic Minesweeper using Python with Tkinter for the GUI.

In an effort to both familiarize myself with Python and Tkinter and to create a simple puzzle game for my sudoku-loving girlfriend to play, I decided on writing a version of Minesweeper, seeing as the default Tkinter GUI looked very reminiscent of the classic look.
You can set the number of spaces on both the horizontal and vertical axis, as well as the number of mines. When you lose, you have the option to continue in "Baby Mode", where the board turns blue and you can continue after hitting a mine.
